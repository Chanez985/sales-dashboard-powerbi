# Sales Dashboard using Power BI

## Project Overview

This project presents an interactive sales dashboard developed with Power BI.

The dashboard provides a visual analysis of sales performance, customer orders, product categories, regional performance, and cancelled orders.

## Objective

The main objective of this project is to build an interactive business intelligence dashboard that helps monitor sales performance and support decision-making.

## Dashboard Pages

The Power BI report includes the following pages:

- **Suivi ventes**: global sales performance, key indicators, filters, sales trends, and regional analysis.
- **Ventes annulées**: analysis of cancelled orders by region, product, and time period.
- **Infobulle région**: custom tooltip for regional insights.
- **Infobulle catégorie produit**: custom tooltip for product-level analysis.

## Key Performance Indicators

The dashboard includes the following KPIs:

- Total sales;
- Number of orders;
- Quantity sold;
- Average order value;
- Sales by customer;
- Sales by region;
- Cancelled orders analysis.
